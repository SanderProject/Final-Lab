# Config Files

This repository contains the start-up configuration files of my final group lab experience.

## Structure

**acc1-config** - contains start-up configuration file of the first layer 2 access switch 
**acc1-config** - contains start-up configuration file of the second layer 2 access switch 
**distri1-config** - contains start-up configuration file of the first distribution layer 3 switch 
**distri2-config** - contains start-up configuration file of the first distribution layer 3 switch 
**RTR1-config* - contains the RTR1 router Start-up config
**infrasw-config** - contains the infrasw Layer 2 switch Start-up config

## Notes

Sensitive data is not included in this repository.
