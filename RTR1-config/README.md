# RTR1

This folder contains the RTR1 router Start-up config

## Features

Layer 2 (Data Link Layer) Protocols
- IEEE 802.1Q (VLAN Tagging / Router-on-a-stick)
- Native VLAN
- DHCP Snooping
- ARP Inspection
- DHCP Snooping
- Port Security

Layer 3 (Network Layer) Protocols
- Open Shortest Path First (OSPFv2)
- IP Routing (IPv4)
- DHCP Relay (IP Helper)
- SSH (Secure Shell)
