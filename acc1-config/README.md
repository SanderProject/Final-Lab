# acc1-Sw

This folder contains the acc1 Access Layer 2 switch Start-up config

## Features

Layer 2 (Data Link Layer) Protocols
- Spanning Tree Protocol (Rapid-PVST)
- VLANs (IEEE 802.1Q)
- EtherChannel (LACP)
- Port Security
- DHCP Snooping
- Dynamic ARP Inspection (DAI)

Layer 3 (Network Layer) Protocols
- Default Gateway
- IP Routing (IPv4)
- NTP (Network Time Protocol)
- SSH (Secure Shell)
- HTTP/HTTPS
