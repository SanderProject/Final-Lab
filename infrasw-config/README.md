# infrasw

This folder contains the infrasw Layer 2 switch Start-up config

## Features

Layer 2 (Data Link Layer) Protocols
- VLANs (IEEE 802.1Q)
- Spanning Tree Protocol (Rapid-PVST)
- DHCP Snooping
- Dynamic ARP Inspection (DAI)
- Port Security
- Switchport Nonegotiate

Layer 3 (Network Layer) Protocols
- Default Gateway
- IP Routing (IPv4)
- HTTP/HTTPS
- SSH (Secure Shell)
