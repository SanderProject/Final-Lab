# PFSense

This folder contains the PFSense Backup

## Features

**Firewall & Netwerkbeveiliging**
- Stateful Packet Inspection (SPI)
- NAT (Network Address Translation)
- Bogons & Private Networks Filtering

**Netwerkdiensten**
- DHCP Server
- DNS Resolver (Unbound)
- Gateway Monitoring

**Gebruikersbeheer & Toegang**
- WebGUI (HTTPS)
- User Authentication
- SSH Toegang

**Monitoring & Logging**
- System Logging (Syslog)
- RRD Graphs
